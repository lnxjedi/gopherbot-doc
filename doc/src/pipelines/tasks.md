# Included Tasks

Gopherbot ships with a small set of useful stock tasks. Common examples include:

- `send-message`
- `restart-robot`
- `robot-quit`
- `pause`
- `pause-brain`
- `resume-brain`
- `rotate-log`
- `tail-log`
- `ssh-agent`
- `ssh-git-helper`
- `git-command`
- `email-log`

These tasks are useful both directly and as examples of how the pipeline model is meant to be used.
